# RepoMind AMD

RepoMind AMD is a specialized dashboard and agentic workflow system designed for advanced repository management and AI-driven development insights.

## Project Structure

- `apps/web`: The Next.js web application providing the dashboard interface.
- `docs/`: Technical specifications and design documentation.
- `lib/`: Shared utilities for AI planning, git storage, and authentication.

## Tech Stack

- **Framework**: Next.js (TypeScript)
- **Styling**: Vanilla CSS
- **Database/Auth**: Supabase & NextAuth.js
- **AI**: Custom agentic workflows for matching and planning.

## Getting Started

1. Navigate to `apps/web`
2. Install dependencies: `npm install`
3. Set up your `.env.local` based on `.env.local.example`
4. Run the development server: `npm run dev`
